# Flowglad Skills

AI agent skills for integrating [Flowglad](https://flowglad.com) billing into your application.

## Installation

```bash
npx skills add flowglad/skills
```

## Available Skills

| Skill | Description |
|-------|-------------|
| `setup` | Install and configure Flowglad SDK |
| `feature-gating` | Implement feature access checks |
| `checkout` | Create purchase and checkout flows |
| `usage-tracking` | Track usage for metered billing |
| `subscriptions` | Manage subscription lifecycle |
| `pricing-ui` | Build pricing pages and displays |

## Supported Agents

- Claude Code
- Cursor
- Windsurf
- Codex
- Other skills.sh compatible agents
