## Overview

Flowglad makes it simple to fetch pricing models and product catalogs so you can present clear plans to your customers. This guide covers the core APIs, SDK helpers, and recommended UI patterns.

## What you'll learn

- Fetch pricing models and products from the Flowglad Server SDK.
- Display product tiers with prices and feature highlights in your frontend.
- Keep your catalog up to date using real-time sync strategies.

## Next steps

- Start with the `displayPricingModel` example in the Server SDK docs.
- Explore UI components in the React and Next.js SDKs for rendering product cards.
