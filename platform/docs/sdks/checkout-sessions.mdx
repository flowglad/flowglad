## Overview

Checkout sessions allow you to hand off payment collection to Flowglad-hosted or embedded flows. Use them to reduce PCI scope while offering a polished purchase experience.

## What you'll learn

- Create checkout sessions with the Server SDK.
- Redirect or embed checkout flows from your frontend SDK.
- Customize session metadata to track plans, coupons, and post-checkout actions.
