## Overview

Accessing a customer's purchase history lets you tailor onboarding, upsell paths, and support flows. Flowglad aggregates past orders alongside active subscriptions for easy consumption.

## What you'll learn

- Fetch purchase history using Server SDK helpers.
- Hydrate frontends with active subscriptions to customize UI.
- Combine purchase data with feature checks for granular entitlements.

## Next steps

- Add a `getCustomerPurchases` call in your account dashboard route.
- Use the React SDK billing context to power personalized landing pages.
